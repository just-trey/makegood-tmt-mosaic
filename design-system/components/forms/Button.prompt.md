A clickable action button, in the app's two variants: neutral "default" and high-emphasis "primary" (teal fill).

```jsx
<Button variant="primary" onClick={doExport}>Export STL set (.zip)</Button>
<Button size="small">Load sample artwork</Button>
```

Variants: `default` (bordered, transparent-ish panel fill) and `primary` (solid teal, dark text). Sizes: `default` and `small`. Supports `disabled` and `fullWidth`.
